# Browser Homepage

## Description:
A browser homepage that serves as a personal dashboard. It features a clean interface with quick access to bookmarks, a search function, and real-time weather information.

## Features:
- Customizable bookmarks organized into categories
- Google search integration
- Real-time weather display for a specified location
- Responsive design for various screen sizes

## Usage
- Set this page as your [Browser's Homepage](https://prajjawal-kansara.github.io/Browser-homepage/) or new tab page to quickly access your frequently used links and information.
- Use the search bar to perform Google searches directly from the homepage.
- View current weather information at a glance.
